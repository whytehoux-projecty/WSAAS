## üìä IMAGE OPTIMIZATION GUIDELINES

### Format Standards

- **Photos**: WebP with JPG fallback
- **Icons**: SVG (vector, scalable)
- **Logos**: SVG primary, PNG fallback
- **Illustrations**: SVG preferred

### Compression

- **WebP Quality**: 85%
- **JPG Quality**: 80%
- **PNG**: Optimized with tinypng/imageoptim

### Responsive Sizes

- **Hero Images**: Provide 1920w, 1280w, 640w versions
- **Cards**: Provide 800w, 400w versions
- **Icons**: SVG (scalable)

### Naming Convention

```
[category]-[description]-[variant].[ext]

Examples:
- hero-personal-banking-desktop.webp
- icon-checking-account-outline.svg
- card-virtual-design-gradient.png
```
